# Retail Sales Performance Dashboard (Excel)

## Overview
This project presents an interactive Excel dashboard analyzing retail sales performance across different states, product categories, and store managers.

The dashboard provides insights into sales trends, regional performance, and category distribution using Excel's data analysis and visualization features.

## Dashboard Preview
![Dashboard Preview](images/dashboard-preview.png)

## Features
- Interactive slicers for Financial Year, State, and Category
- KPI cards displaying Total Sales, Average Sales, and Top Category
- Sales trend analysis using line charts
- Category-wise and manager-wise performance analysis
- Geographic sales visualization using map charts
- State-level trend analysis using sparklines

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Sparklines
- Map Charts
- Slicers

## Dataset
Retail transaction dataset containing information about:
- Sales amount
- Product categories
- Store managers
- Geographic regions
- Time period (2016–2017)

## Key Insights
- Identified top performing product categories
- Compared sales between different retail chains
- Analyzed regional sales performance across Australian states
